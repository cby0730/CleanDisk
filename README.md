# CleanDisk 📱

一個簡潔而強大的 macOS 磁碟空間分析工具，幫助您視覺化磁碟使用情況並輕鬆清理不需要的檔案。

## ✨ 功能特色

### 🔍 智慧掃描
- **快速掃描**：高效掃描任意資料夾或整個磁碟
- **實時進度**：顯示詳細的掃描進度和處理狀態
- **深度分析**：計算真實的磁碟佔用空間（包含稀疏檔案）

### 📊 視覺化展示
- **樹狀結構**：直觀的檔案系統樹狀顯示
- **大小視覺化**：彩色進度條顯示檔案/資料夾相對大小
- **百分比顯示**：清楚顯示每個項目佔用的磁碟空間百分比
- **檔案類型圖示**：不同檔案類型使用不同顏色和圖示

### 🔎 搜尋與瀏覽
- **即時搜尋**：快速找到特定檔案或資料夾
- **搜尋高亮**：搜尋結果自動高亮顯示
- **展開/收合**：可展開或收合資料夾查看詳細內容
- **遞迴顯示**：樹狀結構支援任意深度的資料夾層級

### 📋 詳細資訊
- **基本資訊**：檔案名稱、類型、大小、路徑、修改時間
- **目錄統計**：資料夾內檔案和子資料夾數量統計
- **最大項目**：快速識別資料夾中最大的檔案或子資料夾

### 🗑️ 安全刪除
- **拖放刪除**：將檔案拖拉到刪除區域
- **批次處理**：支援一次選擇多個檔案進行刪除
- **安全確認**：刪除前顯示確認對話框
- **垃圾桶刪除**：檔案移動到垃圾桶，可以復原

### 🎨 使用者體驗
- **原生介面**：使用 SwiftUI 打造的現代化 macOS 介面
- **右鍵選單**：在 Finder 中顯示、展開/收合、複製路徑
- **拖放操作**：支援拖放檔案到刪除區域
- **深色模式**：完整支援 macOS 深色模式
- **記憶體管理**：掃描完成後可清除結果釋放記憶體
- **效能最佳化**：背景執行緒掃描，不阻塞 UI

## 🖥️ 系統需求

- **作業系統**：macOS 15.5 (Sequoia) 或更新版本
- **架構**：支援 Apple Silicon (M1/M2/M3/M4) Mac
- **記憶體**：建議 8GB RAM 以上
- **磁碟空間**：約 50MB 安裝空間

## 📦 安裝方法

> **重要提醒**：此應用程式需要 macOS 15.5 (Sequoia) 或更新版本。如果您的 Mac 版本較舊，請先升級作業系統。

### 從原始碼編譯

**前置需求**：
- macOS 15.5+ 開發環境
- Xcode 16.0 或更新版本
- Command Line Tools

1. **克隆專案**
   ```bash
   git clone https://github.com/cby0730/CleanDisk.git
   cd CleanDisk
   ```

2. **開啟 Xcode**
   ```bash
   open CleanDisk.xcodeproj
   ```

3. **編譯執行**
   - 選擇目標設備（Mac）
   - 按下 ⌘+R 或點擊「Run」按鈕
   - 首次執行可能需要授予檔案存取權限

### 下載編譯版本

1. 前往 [Releases 頁面](https://github.com/cby0730/CleanDisk/releases/tag/installer)
2. 下載最新版本的 `CleanDisk.dmg`
3. 開啟 DMG 檔案並將 CleanDisk.app 拖拉到應用程式資料夾

## 🚀 使用方法

### 基本操作

1. **開始掃描**
   - 啟動 CleanDisk
   - 在左側面板輸入要掃描的路徑（預設為 `/`）
   - 點擊「選擇」按鈕選擇特定資料夾（可選）
   - 點擊「開始掃描」按鈕

2. **瀏覽檔案**
   - 掃描完成後，檔案樹會顯示在中央區域
   - 點擊資料夾旁的箭頭可展開/收合
   - 點擊任何項目可在右側查看詳細資訊

3. **搜尋檔案**
   - 在檔案樹上方的搜尋框輸入關鍵字
   - 符合條件的項目會自動高亮顯示
   - 點擊「清除」按鈕清除搜尋

4. **刪除檔案**
   - 將不需要的檔案或資料夾拖拉到下方的刪除區域
   - 檢查待刪除列表和總大小
   - 點擊「確認刪除」按鈕
   - 在確認對話框中點擊「刪除」

### 進階技巧

- **右鍵選單**：在檔案項目上右鍵可以在 Finder 中顯示、複製路徑等
- **快速分析**：關注紅色和橙色的進度條，這些通常是佔用空間較大的項目
- **批次刪除**：可以將多個項目拖拉到刪除區域後一次刪除
- **路徑檢視**：在詳細資訊面板中可以複製完整的檔案路徑

## 🎯 使用場景

### 磁碟清理
- 找出佔用大量空間的大檔案和資料夾
- 清理下載資料夾中的舊檔案
- 刪除不需要的應用程式和快取檔案

### 開發者工具
- 分析專案資料夾的大小分布
- 清理 build 產出和暫存檔案
- 找出意外的大檔案（如日誌檔案）

### 系統維護
- 定期檢查系統磁碟使用情況
- 在升級前釋放磁碟空間
- 找出重複或遺忘的檔案

## 🛡️ 安全性與隱私

- **只讀掃描**：掃描過程不會修改任何檔案
- **垃圾桶刪除**：所有刪除操作都會將檔案移動到垃圾桶，可復原
- **確認對話框**：刪除前會顯示詳細的確認資訊
- **權限檢查**：遵循 macOS 的檔案權限和隱私保護系統
- **本地處理**：所有數據處理均在本機進行，不上傳任何資訊
- **沙盒環境**：應用程式運行在 macOS 沙盒環境中，保障系統安全
